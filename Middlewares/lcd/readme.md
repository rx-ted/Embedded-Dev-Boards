## 介绍

本项目提供了对嵌入式开发板上LCD显示屏的中间件支持，旨在简化显示屏的驱动和使用。

## 支持的驱动

目前支持以下驱动：

- [X] SSD1306 IIC
- [X] SSD1306 SPI

其他驱动如ST7735，暂时不支持。

- [ ] ST7735
- [ ] 待定
